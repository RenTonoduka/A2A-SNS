# /compare コマンド

複数アカウントのポストを比較分析する

## 使用方法

```
/compare @user1 @user2              # 2アカウント比較
/compare @user1 @user2 @user3       # 3アカウント比較
/compare --files file1.json file2.json  # 既存データで比較
```

## 実行手順

### Step 1: データ準備

指定アカウントのポストを順次抽出:

```bash
python post_extractor.py user1 --max 100
python post_extractor.py user2 --max 100
```

または既存の抽出データを使用。

### Step 2: 比較分析

以下の観点で比較:

1. **エンゲージメント比較**
   - 平均いいね数
   - 平均RT数
   - エンゲージメント率

2. **投稿頻度比較**
   - 日あたり投稿数
   - 最もアクティブな時間帯

3. **コンテンツ傾向比較**
   - ポストの長さ
   - メディア使用率
   - よく使うキーワード

4. **バズ率比較**
   - バズポストの割合
   - 最大バズの規模

---

## 保存先

```
data/analysis/YYYYMMDD_compare_user1_user2.md
```

---

## オプション

| オプション | 説明 |
|-----------|------|
| `--max N` | 各アカウントの抽出件数 |
| `--period 7d` | 期間指定（7d, 30d, 90d） |
| `--export csv` | CSVで統計出力 |

---

## 入力例

```
/compare @naval @pmarca --max 100
```
