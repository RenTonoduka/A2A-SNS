## Step 1: 現状分析

### 既存エージェント構成

```
現在のエージェント（11個）:
├─ research-agent          # 競合動画分析
├─ trend-researcher-agent  # 外部トレンド収集
├─ video-concept-agent     # 台本コンセプト生成
├─ title-strategist-agent  # タイトル戦略
├─ script-reviewer-agent   # 台本評価
├─ script-improver-agent   # 台本改善
├─ pipeline-coordinator-agent  # パイプライン統括
├─ organization-improver-agent # 組織改善
├─ agent-factory-agent     # エージェント生成
├─ coordinator-agent       # 汎用コーディネーター
└─ その他（codegen, deployment, pr, issue, review）
```

### 分析結果

パイプラインを分析した結果、以下のギャップを検出:

1. **フック専門エージェントがない**
   - video-concept-agentに責務が集中
   - フック力スコアが低くなりがち

2. **数値データ付与エージェントがない**
   - 具体性スコア（20点満点）の向上が困難
   - 「月27時間節約」等の具体的数値がない

3. **差別化分析エージェントがない**
   - 競合との違いを明確にする専門家がいない
   - 差別化スコア（15点満点）の向上が困難

4. **CTA最適化エージェントがない**
   - CTAの配置・文言を専門的に設計するエージェントがない

## Step 2: ギャップ検出

| # | ギャップ | 影響スコア軸 | 優先度 | 解決策 |
|---|---------|-------------|--------|--------|
| 1 | フック専門家不在 | フック力 25点 | 高 | hook-specialist-agent 作成 |
| 2 | 数値データ不足 | 具体性 20点 | 高 | data-enrichment-agent 作成 |
| 3 | 差別化分析なし | 差別化 15点 | 中 | differentiation-agent 作成 |
| 4 | CTA最適化なし | CTA 15点 | 中 | 次サイクル |

---

## Step 3: 改善計画

### 今サイクルで作成するエージェント

1. **hook-specialist-agent**
   - 役割: 視聴者を3秒で掴むフック文を専門生成
   - 入力: テーマ、ターゲット、参考タイトル
   - 出力: フック文3案 + 推奨フック

2. **data-enrichment-agent**
   - 役割: 台本に具体的な数値データを付与
   - 入力: 台本ドラフト、テーマ
   - 出力: 数値付き台本、追加した数値リスト

3. **differentiation-agent**
   - 役割: 競合動画との差別化ポイントを設計
   - 入力: テーマ、競合動画リスト
   - 出力: 差別化戦略、独自の切り口
