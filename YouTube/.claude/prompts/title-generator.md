# YouTubeタイトル生成プロンプト v2.0

## 目的

コンテンツのテーマから視聴者の深層心理を分析し、**思考ベース**で複数の戦略パターンからタイトルを生成する。

---

## 入力変数

**{{CONTENT_THEME}}** - コンテンツのテーマ・内容
**{{KEYWORDS}}** - 含めたいキーワード（ツール名、技術名など）※省略可
**{{TARGET_AUDIENCE}}** - ターゲット視聴者層（初心者/中級者/経営者 等）※省略可

---

## Phase 1: 深層心理分析（思考フェーズ）

### 1.1 視聴者が信じている「常識」を特定

このテーマについて、視聴者が当たり前だと思っていることを5つ抽出：

```markdown
| # | 常識（視聴者が信じていること） | なぜそう信じているか |
|---|------------------------------|---------------------|
| 1 | | |
| 2 | | |
| 3 | | |
| 4 | | |
| 5 | | |
```

### 1.2 各常識の「盲点」を発見

各常識に対して「それが成り立つ前提条件」「見落としている真実」を問う：

```markdown
| # | 常識 | 盲点（見落としている真実） | なぜ盲点なのか |
|---|------|--------------------------|---------------|
| 1 | | | |
| 2 | | | |
| 3 | | | |
| 4 | | | |
| 5 | | | |
```

### 1.3 視聴者の深層心理マップ

#### 恐怖（3つ）
視聴者がこのテーマに関して抱えている恐怖：
- 恐怖1: [具体的な恐怖] → 心の声「...」
- 恐怖2:
- 恐怖3:

#### 痛み（3つ）
視聴者が現在感じている痛み・フラストレーション：
- 痛み1: [具体的な痛み] → 影響「...」
- 痛み2:
- 痛み3:

#### 願望（3つ）
視聴者が本当に手に入れたいもの：
- 願望1: [具体的な願望] → 達成後の姿「...」
- 願望2:
- 願望3:

### 1.4 本質の特定

**このコンテンツの「本質」は何か？**

視聴者が表面的に求めているもの（ツールの使い方、テクニック）の奥にある、本当に重要なことは何か？

```markdown
表面的なニーズ: {{表面的に求めているもの}}
         ↓ なぜ？
中間ニーズ: {{その理由}}
         ↓ なぜ？
本質: {{最も重要な1つのこと}}
```

**本質を一言で表すと**: 「〇〇は△△が9割」形式で表現

## Phase 2: 5戦略タイトル生成（思考ベース）

### 戦略1: 逆張り型（常識の盲点を突く）

**生成ロジック**:
Phase 1.2で発見した「盲点」を、視聴者がクリックしたくなる形に言語化

**思考プロセス**:
1. 最もインパクトのある盲点を選択
2. 「〇〇するな。△△しろ」「〇〇の正体は△△だった」などの形で表現
3. 視聴者が「え？なんで？」と思う意外性を確保

**生成タイトル（3案）**:
```markdown
### 逆張り1
**タイトル**:
**突いている盲点**:
**クリック心理**: なぜ視聴者はこれをクリックしたくなるか

### 逆張り2
**タイトル**:
**突いている盲点**:
**クリック心理**:

### 逆張り3
**タイトル**:
**突いている盲点**:
**クリック心理**:
```

### 戦略2: 本質訴求型（「〇〇は△△が9割」）

**生成ロジック**:
Phase 1.4で特定した「本質」を、シンプルで記憶に残る形で表現

**思考プロセス**:
1. 本質を「〇〇は△△が9割」形式で表現
2. 「△△」は意外性があり、かつ真実であること
3. 視聴者が「確かに...」と納得できること

**パターンバリエーション**:
- 「〇〇は△△が9割」
- 「〇〇で成功する人は△△が違う」
- 「〇〇の本質は△△だった」
- 「結局、〇〇は△△で決まる」

**生成タイトル（3案）**:
```markdown
### 本質1
**タイトル**:
**訴求している本質**:
**なぜこれが本質か**:
```

### 戦略3: ベネフィット型（具体的な数字で訴求）

**生成ロジック**:
Phase 1.3の「願望」を、具体的な数字・成果で表現

**思考プロセス**:
1. 視聴者が得られる具体的なベネフィットを特定
2. 可能な限り数値化（時間、お金、効率）
3. 「この動画を見れば〇〇が手に入る」と明確に伝える

**数値化の観点**:
- 時間: 「月27時間節約」「作業時間1/3」
- お金: 「外注費10万円削減」「月収+5万円」
- 効率: 「3倍速」「エラー90%削減」
- 成果: 「初月で〇〇達成」「30日で〇〇」

**生成タイトル（3案）**:
```markdown
### ベネフィット1
**タイトル**:
**訴求ベネフィット**:
**数値の根拠**:

**タイトル**:
```

### 戦略4: 好奇心喚起型（意外性・ギャップ）

**生成ロジック**:
視聴者の予想を裏切る「意外な事実」「ギャップ」で興味を引く

**思考プロセス**:
1. このテーマで「意外」なことは何か？
2. 「〇〇なのに△△」というギャップを作る
3. 視聴者が「どういうこと？」と思う謎を提示

**パターンバリエーション**:
- 「〇〇なのに△△できた理由」（属性ギャップ）
- 「〇〇を使わずに△△する方法」（手段ギャップ）
- 「〇〇の真実を話します」（暴露系）
- 「〇〇したら△△になった」（結果ギャップ）

**生成タイトル（3案）**:
```markdown

**タイトル**:
```

### 戦略5: 緊急性型（今見るべき理由）

**思考プロセス**:
1. このテーマを「今」学ぶべき理由は何か？
2. 見ないとどうなるか（損失回避）
3. 時間的な制約や変化を示す

**パターンバリエーション**:

- 「まだ〇〇してるの？」（遅れへの警告）
- 「〇〇が変わる前に知っておくべきこと」（変化の予兆）

## Phase 3: 最終評価と推薦

### 評価基準

各タイトルを以下の5軸で評価（各5点満点）:

| 評価軸 | 説明 |
|--------|------|
| **クリック欲求** | 視聴者が「見たい」と思うか |
| **本質性** | 表面的でなく、本質を突いているか |
| **具体性** | 抽象的でなく、具体的か |
| **差別化** | 競合と明確に違うか |
| **実現性** | 動画内容として実現できるか |
