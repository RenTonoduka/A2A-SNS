# /trend-research コマンド

外部ソース（X/Twitter、Web、海外）から最新トレンドを収集する

## 使用方法

```
/trend-research [テーマ]
```

## 実行手順

このコマンドは `trend-researcher-agent` を起動し、YouTube以外の外部ソースからトレンド情報を収集します。

---

### Step 1: X/Twitter リサーチ

**使用ツール**: `mcp__exa__web_search_exa` または `WebSearch`

**検索内容**:
```
site:twitter.com OR site:x.com "[テーマ]" バズ
site:twitter.com "[テーマ]" min_faves:1000
```

**抽出項目**:
- バズツイート（いいね1000以上）
- 議論されているトピック
- インフルエンサーの発言

---

### Step 2: Web 最新記事リサーチ

**使用ツール**: `mcp__exa__web_search_exa`

**検索内容**:
```
[テーマ] 2025年 最新
[テーマ] 事例 活用
```

**抽出項目**:
- 直近1週間の注目記事
- 新しいツール・サービス
- ユースケース・事例

---

### Step 3: 海外トレンドリサーチ

**検索内容**:
```
site:news.ycombinator.com [テーマ英語]
site:reddit.com/r/artificial [テーマ英語]
```

**抽出項目**:
- Hacker News で話題のツール
- Reddit での議論
- 日本未上陸のサービス

---

### Step 4: トレンド分析

**分析内容**:
1. 共通キーワードの抽出
2. 盛り上がり度のスコアリング
3. 動画ネタ化の可能性評価
4. 競合YouTuberのカバー状況確認

---

### Step 5: 企画提案

**出力内容**:
- 動画企画案 3〜5本
- 各企画の差別化ポイント
- 推奨タイトル案
- 投稿推奨タイミング

---

## 出力

```
research/trends/YYYYMMDD_[テーマ].md
```

---

## 連携

このコマンドの出力は以下で使用:

- `/pipeline` → トレンド情報としてvideo-concept-agentに渡される
- `/video-concept` → 最新トレンドを踏まえた台本生成

---

## 注意事項

- Web検索には `mcp__exa__web_search_exa` または `WebSearch` ツールを使用
- 検索結果が少ない場合は検索クエリを調整
- 英語検索と日本語検索を両方実行して網羅性を確保
