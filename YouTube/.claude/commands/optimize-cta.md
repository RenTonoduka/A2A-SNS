# /optimize-cta コマンド

台本のCTA（Call To Action）を最適化

## 使用方法

```
/optimize-cta [台本ファイルパス]
/optimize-cta                    # 最新の台本を自動選択
```

## 実行手順

このコマンドは `cta-optimizer-agent` を起動し、CTAの配置・表現・タイミングを最適化します。

---

### Step 1: 現状CTA分析

**分析項目**:
- 既存CTAの数と配置
- 表現パターン
- 価値提供との連動

---

### Step 2: CTA設計

**最適化ポイント**:
1. **冒頭CTA**: 最後まで視聴する動機づけ
2. **中盤CTA**: エンゲージメント確保（いいね）
3. **終盤CTA**: 本命アクション（登録）

---

### Step 3: 台本への反映

**出力内容**:
- CTA最適化レポート
- Before/After比較
- スコア予測

---

## 出力例

```
## CTA最適化

### Before
- CTA数: 1箇所（終盤のみ）
- 表現: 「チャンネル登録お願いします」
- 問題: 理由なし、唐突

### After
- CTA数: 3箇所（冒頭・中盤・終盤）
- 表現: 理由付き + 仲間意識

### 追加CTA

**中盤（5:00付近）**:
> 「ここまでの内容、役立ちましたか？
> いいねボタンで教えてください」

**終盤（改善版）**:
> 「次回は応用テクニックを紹介します。
> 見逃さないよう、チャンネル登録を」

### スコア予測
CTA力: 8/15 → 13/15（+5点）
```

---

## 連携

- 入力: `/video-concept` または `/script-improve` の台本
- 出力: `/script-review` でCTA力スコアが向上
