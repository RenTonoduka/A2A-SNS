# /hook-generate コマンド

視聴者を3秒で掴むフック文を専門生成

## 使用方法

```
/hook-generate [テーマ] [ターゲット]
```

## 実行手順

このコマンドは `hook-specialist-agent` を起動し、5つのテクニックからフック文を生成します。

---

### Step 1: ターゲット心理分析

**分析項目**:
- 痛み: 今抱えている問題
- 願望: 達成したいこと
- 恐怖: 避けたいこと
- 好奇心: 知りたいこと

---

### Step 2: フック生成

**5つのテクニック**:
1. 衝撃の数字型: 「月27時間節約」
2. 問いかけ型: 「まだ〇〇してるの？」
3. 逆説型: 「〇〇を学ぶな」
4. 衝撃の事実型: 「〇〇は3%しかいない」
5. ストーリー導入型: 「半年前まで僕は...」

---

### Step 3: 評価・選定

**評価軸**:
- 注意喚起力: /10
- 続きを見たい度: /10
- ターゲット適合度: /10

TOP3を推薦理由とともに出力。

---
