# /design-structure コマンド

台本の構成を設計

## 使用方法

```
/design-structure [テーマ] [動画長さ]
/design-structure AIエージェント 15分
```

## 実行手順

このコマンドは `structure-designer-agent` を起動し、フレームワークを適用した構成を設計します。

---

### Step 1: フレームワーク選択

**選択肢**:
- **PASTOR**: 問題解決型（チュートリアル向け）
- **AREA**: 主張型（意見・解説向け）
- **ハイブリッド**: 組み合わせ

---

### Step 2: 構成設計

**設計項目**:
- タイムスタンプ付き構成表
- 各セクションのキーメッセージ
- 離脱防止パターン

---

### Step 3: アウトライン生成

**出力内容**:
- 全体構成図
- セクション詳細
- 感情曲線

---

## 出力例

```
## 構成設計

### フレームワーク: PASTOR + オープンループ

### 全体構成

[0:00] フック（衝撃の数字）
[0:30] Problem（痛みへの共感）
[1:30] Amplify（問題の深刻化）
[3:00] Story（失敗体験）
[5:00] Transformation（転機）
[7:00] Offer（解決策）
[13:00] Response（CTA）

### 離脱防止

| 時間 | 対策 |
|------|------|
| 3分 | ★価値提供1 |
| 6分 | ★価値提供2 |
| 9分 | ★価値提供3 |
```

---

## 連携

- 入力: テーマ、ターゲット情報
- 出力: `/video-concept` の構成として使用
